## P0: プロジェクトセットアップ

### 目的

プロジェクトの骨格を構築し、デザインシステム（カラー、フォント、スペーシング）をCSS変数として定義する。

### 作業内容

1. ディレクトリ構造を作成:
   ```
   /
   ├── index.html
   ├── css/
   │   └── style.css
   ├── js/
   │   └── main.js
   └── public/
       └── images/    （既存）
   ```

2. `index.html` に基本構造を作成:
   - `<!DOCTYPE html>` + lang="ja"
   - meta charset, viewport
   - Google Fonts 読込（Shippori Mincho, Noto Sans JP）
   - CSS/JS リンク
   - 各セクションの空 `<section>` タグ（id付き）

3. `css/style.css` にデザインシステムを定義:
   - CSS カスタムプロパティ（カラーパレット、フォント、スペーシング）
   - リセットCSS（box-sizing, margin/padding reset）
   - 基本タイポグラフィ
   - コンテナ幅（max-width: 1200px）

### 参照仕様書
- `20260312-105216-homepage-spec.md` — デザインガイドライン、カラーパレット、フォント

### 完了条件

   - システムカードの交互スライドイン

4. **画像最適化**:

   - lazy loading（loading="lazy"）
